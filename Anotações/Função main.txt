1. Conceito Central: Interface de Execução

A função main() é o ponto de entrada de todo programa C++. Ela possui uma assinatura especial para receber dados diretamente da linha de comando do sistema operacional.

2. Componentes Rigorosos (Argumentos de Linha de Comando)

A assinatura padrão é: int main(int argc, char const *argv[])

2.1. argc (Argument Count): Um inteiro que conta o número de argumentos passados para o programa. O valor mínimo é sempre 1 (o nome do próprio programa).

2.2. argv (Argument Vector): Um array de strings (ponteiros para char constante) que contém os argumentos.

argv[0] sempre contém o nome (ou caminho) do executável.

Os argumentos fornecidos pelo usuário começam em argv[1].

3. Uso

É obrigatório checar argc antes de tentar acessar qualquer elemento em argv além do argv[0], para evitar falhas de acesso de memória.
