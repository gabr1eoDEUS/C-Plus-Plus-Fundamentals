1. Conceito de Múltiplas Condições
Intenção: Controlar o fluxo de execução para decisões que envolvem três ou mais opções (não apenas A ou B).

Como funciona: O programa testa as condições em sequência.

2. Sintaxe Rigorosa
Início: Começa com um bloco if para a primeira condição.

Caminhos Intermediários: Segue com um ou mais blocos else if para as condições seguintes.

O else if só é avaliado se todas as condições if e else if anteriores forem false.

Caminho Padrão (Opcional): Termina com um bloco else opcional, que é o caminho de fallback (executado se absolutamente nenhuma condição for true).

3. Rigidez Crucial: Exclusividade
Regra: Em uma cadeia if / else if / else, apenas UM ÚNICO BLOCO de código será executado.

Sequência: Assim que uma condição resulta em true, o bloco correspondente é executado e o programa pula (ignora) todas as verificações else if e o else restante, continuando o código após a estrutura.

Recomendação: Use if / else if para testar faixas de valores (por exemplo, nota A, B, C, D) onde a ordem de verificação é importante.
